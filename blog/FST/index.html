FST

06 November, 2019 - 2 min read

The goal was to develop a Mobile App as well as a Web App capable of managing a ski team. The commitment has been given by one of the most prestigious ski teams in Italy, the Falconeri Ski Team. Here are some of the hurdles:

  • manage different groups / categories of skiers & instructors
  • collect enough info for the single workout / race in a simple way
  • single skier profile with some analysis

App Screenshots

Tools

The app was realized using the Ionic Framework, written in Angular. The authentication process was managed using Google's Firebase, the whole backend was recently moved to a proprietary server and developed via SQL.
